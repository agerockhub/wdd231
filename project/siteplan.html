StaffManage Pro - Website Plan Document

1. Site Name

StaffManage Pro

This name reflects the website's purpose of managing teaching staff efficiently. It clearly communicates the system's function and maintains a professional identity suitable for administrative use.

2. Site Purpose

The purpose of StaffManage Pro is to provide a dynamic, lightweight, and user-friendly platform that allows administrators to manage teaching staff information. The system captures, organizes, and retrieves personal and professional staff details, while also tracking current and completed student batches. Built entirely with HTML, CSS, JavaScript, and JSON, this system enhances productivity, supports fast decision-making, and functions without a server-side database.

3. Scenarios

  • How can I quickly find details about a specific staff member, such as their qualifications or assigned classes?
  • Where can I view which student batches are currently active and which ones are already completed?
  • How do I update a teacher's profile or assign them to a new class?

4. Color Scheme

Primary Color: Royal Blue (#1E4DB7) - Used for headings, navigation, and key UI elements.

Secondary Color: Soft Gray (#E5E5E5) - Used as background for clean readability.

Accent Color: Dark Gray (#333333) - Used for paragraph text and labels.

5. Typography

Poppins: Used for all headings to maintain a modern and professional look.
Roboto: Used for body text to ensure high readability and clean presentation.
